Datei Formate

Auf dieser Seite werden Verschiedene DateiFormate miteinander verglichen. Dabei werden Vor und Nachteile herforgehoben wie auch verwendungszwecke dieser Formate genannt.

Wie Hier ganz klahr hervorsticht ist Bitmap das grösste Format. Das liegt ganz Simpel daran, wie ein Bitmap gespeichert wird. Ein Bitmap speichert die Farbe und position jedes einzelnen Pixels ab. Das zweitgrösste Format ist Gif. Das Gif ist vorallem wegen der Farbpalette eingeschränkt. Gifs können aber animiert werden und erlauben so mehrere Bilder in einem File. Als nächstes das Png. Ein Png wird oft für kleinere Grafiken verwendet da es eine hohe Farbauflösung und transparente Pixel unterstützt. Allerdings gibt es aber auch anwendungen die dieses Format nicht unterstützten. Das letzte Format ist das Jpg. Das Jpg wird vorallem dann verwendet wenn grosse Bilder gebraucht werden. Jpg erlaubt es verhältissmässig viele Pixel auf wenig Datenmenge zu komprimieren. Das liegt vorallem daran wie das Jpg Farben speichert. Es Speichert Farben in Blöcken ab (artefakte oben).

Format Grösse Video
MP4 (MPEG-4 Part 14) 5.83 MB
OGG (Operation Good Guys) 8.50 MB
Webm (Web Media) 39.0 KB

Die Videoformate. MP4 ist das wohl bekanteste Videoformat. Bei einem MP4 können verschiedene Dinge priorisiert werden wie Framerate, Auflösung oder Dateigrösse. Das Mp4 unterstüzt zudem auch mehrere integrierte Audio oder Untertietel Spuren. Ogg ist ein Multimedia Dateiformat. So können Videos,Audio oder auch Texte unter einer Fileendung abgespeichert werden. Es ist copyright free und kann so einfach verwendet werden. Webm ist das Mp4 des Internets. Webm ist nach Mp4 das meistverwendetste Audiovideo Format und ist speziell auf Online anwendung ausgelegt.

Format Grösse Sound
MP3 (MPEG audio Layer-3) 20.4KB
ogg (Operation Good Guys) 4.13 KB
Wav 177 KB

MP3 ist das das meist verbreiteste Audioformat. Das Mp3 Format hat viele Vorteile. Neben Kompatibelität mit vielen Geräten und die verlustfreie Komprimierung. Beim exportiren von MP3 files, kann man die Bitrate selber einstellen und so auch bestimmen wie gross und qualitativ hochwertig das File seine soll. Ogg wird neben Video auch für Audio verwendet. Ogg ist leider weniger bekannt und muss vor dem Bearbeiten umgewandelt werden. Dies sorgt auch bei kleiner Dateigrösse dafür das es sich nicht gegen MP3 durchsetzten kann. Als letztes noch Wav. Wav Dateien sind oft sehr gross da das Dateiformat nicht Komprimiert. Deswegen auch lassen sie sich ohne probleme bearbeiten.
